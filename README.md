# SkillFinder

SkillFinder is a React app that helps soccer players find YouTube training videos based on skills, position, and level.

## Features

- Search soccer skills (shooting, passing, etc.)
- Filter by position and level
- Uses YouTube API to fetch training videos

## Tech Stack

- React
- JavaScript
- Vite
- YouTube Data API

## Setup

```bash
npm install
npm run dev
```
